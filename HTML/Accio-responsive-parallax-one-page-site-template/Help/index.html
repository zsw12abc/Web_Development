The Main Features

This template presents very clean and unique design. The most prominent feature of this product is that it is great one-page template with three different Home pages: Layer Slider, Video Background, Slider Parallax. There are also such pages as blog, blog single, shortcodes, project single, project single 2, project single 3.

Great thing is that with this template you are able to use video for pages background, instead of slider and parallax images.

HTML Structure

The structure of this one-page template is divided into few sections. Each sections playes the role of a separate page, thus each section has its own ID which matches Menu Items (but not the same for Sub-menu pages, they are separate pages).These sections have class .page. By the way, you can move through all sections with "up"&"down" keys.

These sections basically initialize the animation starting. Mostly each block can be animated by using one of seven pre-setcanimation effects. For that you only need to add the class for a block: .opacity, .opacity2x, .scale, .slideRight, .slideLeft, .slideDown, .slideUp. Each class speaks and describes the annimation effect by its name. Just for your notes, opacity2x differs from opacity by the speed of appearing, opacity2x is faster.

As it has been mentioned before there are additional single Blog and Project pages except of main index ones.

On the Single Blog Page fixed layout with two columns is used. All of the information within the main content area is nested within a div with an id of "content". The sidebar's (column #2) content is within a div with an id of "sidebar".

But the general template structure on three index pages is the same throughout the template. Here is the general structure.

Any sections on the page can use preset classes

Here are classes for padding :

  1. .padding-top-off - top padding turning off
  2. .padding-bottom-off - bottom padding turning off
  3. .padding-off - top and bottom padding turning off

Class for border:

  1. .border

Class for background:

  1. .bg-gray-color
  2. .bg-turquoise-color
  3. .bg-dark-color

To create parallax effect firstly you need to insert "div" with calss .full-bg-image into section and then define the style with an image for this class.

To add grid to your parallax you need to add "div" with class .parallax-overlay after "div" with class .full-bg-image.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

If you need to change the color, size or font family for the main pages headings or titles of some blocks on pages, you can do this by finding and editing appropriate lines:

  3. jquery.modernizr.js Modernizr is a JavaScript library that detects HTML5 and CSS3 features in the user’s browser.
  4. jquery.flexslider.js For services page
  5. layerslider.kreaturamedia.jquery.js Responsive Slider

  7. jquery.easing.1.3.min.js jQuery easing can make your animations look a lot better.
  8. jquery.gmap.min.js gMap is a jQuery plugin embedding Google Maps into your website.
  9. jquery.mb.YTPlayer youtube video player
  10. jquery.mixitup.js MixItUp is a light-weight but powerful jQuery plugin that provides beautiful animated filtering and sorting of categorized and ordered content.
  11. jquery.queryloader2.jsA simple script to create a preloader for images.

  13. jquery.superslides.min.js Superslides is a full screen, hardware accelerated slider for jQuery.
  14. jquery.touchswipe.min.js A jquery plugin to be used on touch devices such as iPad, iPhone, android etc
