介绍

PHP (PHP Hypertext Preprocessor)的优势:

  • 跨平台兼容
  • 兼容几乎所有服务器
  • 可以支持大多数的数据库
  • 开源免费
  • 文件以.php结尾

PHP能做什么:

  • 生成动态网页
  • 在服务端可以对文件进行多种处理
  • 可以收集信息和数据
  • 可以收发cookies
  • 可以修改维护数据库
  • 可以加密信息

安装

由于PHP是服务器语言, 所以开发者必须要有一个服务器或者本地服务器才能运行PHP语言.

  • 安装一个web server
  • 安装PHP
  • 安装数据库 (e.g.MySQL)

Syntax

PHP可以放在文本的任意地方, 只需要以<?php开头和以?>结尾即可

<?php
// PHP code goes here
?>

PHP文件的扩展名是.php. 一般性包括<html></html>

<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>

  • 方便自己之后的debug和代码复用, 防止忘记代码的作用

Case Sensitivity

PHP不是Case Sensitivity, 这意味着对于关键字(if, while, echo...)来说, 大小写都可以识别, 然而对于设定的函数名来说, 还是Case Sensitivity(区分大小写的)

<?php
ECHO "Hello World!<br>";
echo "Hello World!<br>";
EcHo "Hello World!<br>";
?>

PHP Variables

与别的语言不同, PHP声明variable的时候需要在之前添加符号$

<?php
$txt = "Hello world!";
$x = 5;
$y = 10.5;
?>

variable的命名有要求

  • variable name只能以字母或下划线_开头, 不能以数字开头
  • variable name可以含有数字, 字母和下划线
  • variable name是case sensitive的, 也就是说$hello与$HELLO是两个不同的variable

PHP Output Variable

PHP中用echo来表示输出, echo可以输出variable, 还可以对variable进行简单地处理和计算.

<?php
$txt = "W3Schools.com";
echo "I love " . $txt . "!";
$x = 5;
$y = 4;
echo $x + $y;
?>

PHP Variables Scope

由于PHP可以在一个文档的任意位置, 所以variable的scope就变得很重要了

Global and Local Scope

一个在function外部声明的variable有Global Scope, 可以且仅可以在function外部执行调用.

<?php
$x = 5; // global scope

function myTest() {
    // using x inside this function will generate an error
    echo "<p>Variable x inside function is: $x</p>";
} 
myTest();

echo "<p>Variable x outside function is: $x</p>";
?>

在function内部声明的variable有Local Scope, 可以且仅可以在funciton内部执行

<?php
function myTest() {
    $x = 5; // local scope
    echo "<p>Variable x inside function is: $x</p>";
} 
myTest();

// using x outside the function will generate an error
echo "<p>Variable x outside function is: $x</p>";
?>

PHP The global Keyword

global是PHP中的关键字, 可以让开发者在function内部调用外部的variable

<?php
$x = 5;
$y = 10;

function myTest() {
    global $x, $y;
    $y = $x + $y;
}

myTest();
echo $y; // outputs 15
?>

PHP会把所有的global variable都储存在一个叫$GLOBALS[*index*]的Array中, 其中的index就是variable name. 所以在function内部调用这个Array的话, 不但可以取到variable的值, 还可以对其进行修改

<?php
$x = 5;
$y = 10;

function myTest() {
    $GLOBALS['y'] = $GLOBALS['x'] + $GLOBALS['y'];
} 

myTest();
echo $y; // outputs 15
?>

PHP The static Keyword

在PHP function声明的variable, 在function运行完之后就会被销毁, 但有时候开发者不希望这个variable被销毁, 那么可以用static这个关键字

<?php
function myTest() {
    static $x = 0;
    echo $x;
    $x++;
}

myTest();
myTest();
myTest();
?>

PHP echo and print Statements

在PHP中有两种输出方法, echo和print, 这两个不是函数, 而是语言结构, 用法上基本相同,
最重要的不同点是:

  • echo可以输出多个字符串, 而print只可以输出一个字符串
  • print可以表现得像一个函数, 比如: $result = print "Hello World";
  • print()返回值, 成功返回1, 否则返回0. 而echo()没有返回值
  • echo相对速率比较快, 因为没有返回值

PHP5 Data Type

对于PHP来说, 声明variable的时候不需要声明variable的类型, 然而开发者始终要很清楚的知道这个variable的类型, 因为不同的类型可以做不同的事情.

PHP String

String就是字符串, 写的时候用双引号或者单引号包括进去

<?php 
$x = "Hello world!";

echo $x;
echo "<br>"; 
echo $y;
?>
Get Length of String

PHP中用strlen()来返回string的长度

<?php
echo strlen("Hello world!"); // outputs 12
?>
Count number of words in a String

str_word_count()可以返回string中的单词个数

<?php
echo str_word_count("Hello world!"); // outputs 2
?>
Reverse a String

PHP中用strrev()来颠倒String

<?php
echo strrev("Hello world!"); // outputs !dlrow olleH
?>
Search For a Specific Text Within a String

PHP中提供了一个函数来让开发者可以在一个string中搜索一个特定的字符, strpos()
如果搜索到了, 这函数会返回第一次复合的位置, 如果没有搜索到, 就会返回false

<?php
echo strpos("Hello world!", "world"); // outputs 6
?>
Replace Text Within a String

str_replace()可以用来替换string中的一些字节

<?php
echo str_replace("world", "Dolly", "Hello world!"); // outputs Hello Dolly!
?>

PHP Integer

Integer就是整数, 范围是介于-2147483648 到2147483647之间

  • Integer至少有一位数字
  • Integer不能含有空格或者逗号
  • Integer不能有小数点
  • Integer包含正负数
  • Integer有三种形式:
    • decimal 十进制
    • hexadecimal 十六进制 以0x开头
    • octal 八进制 以0开头
<?php 
$x = 5985;
var_dump($x);
?>

PHP Float

Float是指包含小数点的数

<?php 
$x = 10.365;
var_dump($x);
?>

PHP Boolean

Boolean 只有两个值: true 或者 false

$x = true;
$y = false;

PHP Array

Array是用来存储数据的数据结构, 可以把多个数据存在一个variable中

<?php 
$cars = array("Volvo","BMW","Toyota");
var_dump($cars);
?>
PHP Indexed Arrays

Indexed Arrays的index是自动生成的(从0开始)
开发者可以用index来调用相应的值
用count()来返回array的长度

<?php

echo "I like " . $cars[0] . ", " . $cars[1] . " and " . $cars[2] . ".";
echo count($cars);
?>
PHP Associative Arrays

Associative Arrays中的key是开发者命名的, 没有index, key与value是一一对应的
可以看做是JAVA中的Hashmap

<?php

echo "Peter is " . $age['Peter'] . " years old.";
?>

PHP Object

Object是一种独特的数据结构, 它可以用来存储数据和处理数据
Object必须要被先明确的申明后才能用, 一般性开发者用关键字class来定义一类Object, 然后再用这类Object来创建一个特定的Object

<?php
class Car {
    function Car() {
        $this->model = "VW";
    }
}

// create an object
$herbie = new Car();

// show object properties
echo $herbie->model;
?>

PHP NULL Value

Null在PHP中是一个特殊的类型, 只有一个值是NULL, 代表的意思是: 这个variable没有任何的值
如果一个variable被申明, 但是没有赋值, 那么会被自动赋值为null

<?php
$x = "Hello world!";

PHP5 Constants

PHP中有一些variable一旦被定义了, 它就不可以被修改, 这类variable被称作constant
contant name开头是一个字母或者下划线(没有$), 而且它默认是global的, 哪怕在function外部定义, 在function内部也可以调用

Syntax

Constant可以选择是case-insensitive与否, 默认是false, 也就是说默认区分大小写.

PHP Conditional Statement

PHP if Statement

Syntax
if (condition) {
    code to be executed if condition is true;
} elseif (condition) {
    code to be executed if condition is true;

    code to be executed if condition is false;

PHP Switch Statement

switch Statement是if statement的简写形式
Switch会根据条件来选择进入哪个case, 然后运行到break后跳出statement, 如果都没有满足的条件, 则会运行default

Syntax
switch (n) {
    case label1:
        code to be executed if n=label1;
        break;
    case label2:
        code to be executed if n=label2;
        break;
    case label3:
        code to be executed if n=label3;
        break;
    ...
    default:
        code to be executed if n is different from all labels;
}

PHP Loop

PHP while Loop

while loop会先判定条件, 然后再执行循环, 所以while loop最少执行0次

Syntax
while (condition is true) {
    code to be executed;
}

PHP do…while Loop

do…while loop会先执行循环, 然后再判定条件, 所以do…while loop最少执行1次

Syntax
do {
    code to be executed;
} while (condition is true);

PHP for Loop

for loop的条件判定是由三部分组成的

  • 初始化: 初始化loop counter的值
  • 判定: 判定是否满足循环的条件
  • 结束句: 当运行完一次loop后对loop counter进行处理
Syntax
for (init counter; test counter; increment counter) {
    code to be executed;
}

PHP foreach Loop

foreach loop是for loop的一种变化形式, 只对array适用.
由于开发者时常需要遍历array里面的值, 所以开发者可以用foreach很形象化的去array里面把所有的值一一取出

Syntax
foreach ($array as $value) {
    code to be executed;
}
<?php 
$colors = array("red", "green", "blue", "yellow"); 

foreach ($colors as $value) {
    echo "$value <br>";
}
?>

$_REQUEST, $_POST 和 $_GET的区别

  • $_REQUEST[]具用$_POST[] $_GET[]的功能，但是$_REQUEST[]比较慢。通过POST和GET方法提交的所有数据都可以通过$_REQUEST数组获得。
  • GET是从服务器上获取数据，POST是向服务器传送数据。
  • GET是把参数数据队列加到提交表单的ACTION属性所指的URL中，值和表单内各个字段一一对应，在URL中可以看到。POST是通过HTTP POST机制，将表单内各个字段与其内容放置在HTML HEADER内一起传送到ACTION属性所指的URL地址。用户看不到这个过程。
  • 对于GET方式提交表单数据，服务器端用$_GET[‘name’]获取变量的值，对于POST方式提交表单数据，服务器端用$_POST[‘name’]获取提交的数据，当然，两者都可以通过$_REQUEST[‘name’]获得表单数据。对于REQUEST方式提交表单数据，服务器端用$_REQUEST[‘name’]获取变量的值，但这种方式很少用。
  • GET传送的数据量较小，不能大于2KB。POST传送的数据量较大，一般被默认为不受限制。但理论上，一般认为不能超过100KB。
  • GET安全性非常低，POST安全性较高。
  • GET表单值可以通过_GET获取；但通过action的url设置的参数总是获取不到的，<form method="get" action="a.asp?b=b">跟<form method="get"action="a.asp">是一样的，也就是说，在这种情况下，GET方式会忽略action页面后边带的参数列表。POST表单值可以通过_POST获取；但通过action的url参数设置的参数则可以不能通过_POST获取到。action=test.php?id=1这种就是GET方式传值，可以用$_REQUEST和$_GET接受传值，但不能用POST方式获取到值，即使表单是POST方式提交。所以，在提交表单时，如果action中同时有参数，最好只能通过POST表单方式，对于表单内数据，直接通过POST获取，对于action中参数，通过GET获取。

由于PHP是于服务器沟通的语言, 所以当开发者构建了一个提交到服务器的表单给用户使用的时候, 用户输入的一些数据有可能会影响到服务器的使用, 会造成服务器崩溃或者丢失数据.
所以Form Validation是很必要的

<form method="post" action="<?php echo htmlspecialchars($_SERVER["PHP_SELF"]);?>">

$_SERVER[PHP_SELF]会返回当前执行的script的文件名, 所以$_SERVER[PHP_SELF]会把提交的表单数据返回给页面本身, 而不是跳转到别的页面. 所以用户的误输入或者黑客的蓄意攻击会导致页面的崩溃
htmlspecialchars()这个function是用来把一些特殊字符<,>,/转化成无害的字符&lt, &gt. 这样当用户或者黑客输入一些HTML或者Javascript的代码的时候, 服务器会把这些特殊字符转化成无意义的字符, 然后保存数据. 于是降低了被蓄意攻击的危险.

Validate Form Data With PHP

为了防止用户的误输入或者黑客的攻击导致页面崩溃,

  • 首先, 开发者要先把所有用户输入的数据都经过htmlspecialchars()来把特殊字符转化成无意义的字符
  • 其次, 开发者要删除用户输入的无意义字符(多余的空格, tab, 换行符), 可以使用trim()
  • 最后, 开发者要移除反斜杠符号\, 可以使用stripslashes()

这是一个简单的用来测试用户输入的代码

<?php
// define variables and set to empty values
$name = $email = $gender = $comment = $website = "";

if ($_SERVER["REQUEST_METHOD"] == "POST") {
  $name = test_input($_POST["name"]);
  $email = test_input($_POST["email"]);
  $website = test_input($_POST["website"]);
  $comment = test_input($_POST["comment"]);
  $gender = test_input($_POST["gender"]);
}

function test_input($data) {
  $data = trim($data);
  $data = stripslashes($data);
  $data = htmlspecialchars($data);
  return $data;
}
?>

PHP include与require的区别

include与require都是用来加在外部文件的方法, 区别在于:

  • 如果加在失败:
    • include会产生一个warning, 但是脚本会继续执行
    • warning会爆出一个fatal error, 而脚本会停止执行
    • 可以使用require_once()和include_once()检测文件是否重复包含
  • 对于PHP的性能来说:
    • include执行文件的时候, 每次都需要进行读取和评估
    • require对于文件只处理一次
    • 所以对于需要多次执行一段代码, 使用require的效率比较高.
    • 而如果每次执行代码的文件不同, 或者需要对一个文件的代码进行叠代的循环, 就要使用include
  • 使用方式也不同:
    • 对于require来说, 一般性是放在PHP 文件的最前面, 执行的时候就会先读入require所指定的文件, 变成PHP程序的一部分
    • 对于include来说, 一般是放在流程控制的处理区段中, PHP在执行到include的时候, 才将其文件读出来, 这样可以使流程简单化.

PHP readfile() Function

PHP Open File - fopen()

fopen()包含两个参数, 第一个是要打开文件的名字, 第二个是以什么方式打开

打开方式 解释
r 打开一个文件, 同时只读, 指针在文件的开头
w 打开一个文件, 同时只可以写, 擦除文件的所有内容或者创建一个空的文件, 指针在文件的开头
a 打开一个文件, 同时只可以写, 如果文件存在, 那么不会擦除所有内容, 只会在文件最后开始写入, 如果文件不存在, 就会创建一个新文件
x 新建一个文件, 同时只可以写. 如果文件已经存在则会返回False和error
r+ 打开一个文件, 只读可写. 指针在文件的开头
w+ 打开一个文件, 只读可写. 擦除文件的所有内容或者创建一个空的文件, 指针在文件的开头
a+ 打开一个文件, 只读可写. 如果文件存在, 那么不会擦除所有内容, 只会在文件最后开始写入, 如果文件不存在, 就会创建一个新文件
x+ 新建一个文件, 只读可写. 如果文件已经存在则会返回False和error 
<?php
$myfile = fopen("webdictionary.txt", "r") or die("Unable to open file!");
echo fread($myfile,filesize("webdictionary.txt"));
fclose($myfile);
?>

PHP Write to File - fwrite()

当一个文件已经被打开, 同时可写的时候, 开发者可以使用fwrite() 往文件里面写内容
fwrite()包含两个参数: 第一个是写入的文档名字, 第二个是写入的内容
开发者要注意的是, fopen()里面定义了文件打开的方式, 包括了是擦除所有内容后重写还是在原有内容的结尾继续写. 而fwrite()则只负责按照之前的打开方式写入内容

<?php
$myfile = fopen("newfile.txt", "w") or die("Unable to open file!");
$txt = "John Doe\n";
fwrite($myfile, $txt);
$txt = "Jane Doe\n";
fwrite($myfile, $txt);
fclose($myfile);
?>

PHP5 File Upload

PHP 中上传文件是很容易的, 但是开发者必须很小心用户上传的文件有可能会对数据和服务器造成攻击

Configure The “php.ini” File

首先, 开发者要在php.ini文件中允许用户上传文件.

file_uploads = On

Create The HTML Form

注意:

  • form的提交要用method="post"
  • form还可以添加一些别的属性: enctype="multipart/form-data", 这具体说明了提交form的时候用了哪一种编码形式
  • type="file"是用来上传文件的, 会在输入框边上生成一个选择文件的"Browse"按钮
  • 提交后的form信息会被存储到upload.php文件中.
<!DOCTYPE html>
<html>
<body>

<form action="upload.php" method="post" enctype="multipart/form-data">
    Select image to upload:
    <input type="file" name="fileToUpload" id="fileToUpload">
    <input type="submit" value="Upload Image" name="submit">
</form>

</body>
</html>

Create The Upload File PHP Script

$target_dir = "uploads/"声明了文件上传后存储的位置
$target_file声明了上传文件在用户本地的路径
$imageFileType存储了上传文件的文件扩展名

<?php
$target_dir = "uploads/";
$target_file = $target_dir . basename($_FILES["fileToUpload"]["name"]);
$uploadOk = 1;
$imageFileType = pathinfo($target_file,PATHINFO_EXTENSION);
// Check if image file is a actual image or fake image
if(isset($_POST["submit"])) {
    $check = getimagesize($_FILES["fileToUpload"]["tmp_name"]);
    if($check !== false) {
        echo "File is an image - " . $check["mime"] . ".";
        $uploadOk = 1;
    } else {
        echo "File is not an image.";
        $uploadOk = 0;
    }
}
?>

Check if File Already Exists

在上传文件前, 开发者可以先检查用户是否已经上传了一样的文件到服务器的upload文件夹中. 如果已经上传了, 那么显示提示信息, 同时$uploadOk设为0.

// Check if file already exists
if (file_exists($target_file)) {
    echo "Sorry, file already exists.";
    $uploadOk = 0;
}

Limit File Size and Type

同时, 开发者还可以给于上传文件大小的限制和文件的类型, 如果超出限制或者文件类型不符合, 那么显示提示信息, 同时$uploadOk设为0.

 // Check file size
if ($_FILES["fileToUpload"]["size"] > 500000) {
    echo "Sorry, your file is too large.";
    $uploadOk = 0;
}
// Allow certain file formats
if($imageFileType != "jpg" && $imageFileType != "png" && $imageFileType != "jpeg"
&& $imageFileType != "gif" ) {
    echo "Sorry, only JPG, JPEG, PNG & GIF files are allowed.";
    $uploadOk = 0;
}

完整的代码

<?php
$target_dir = "uploads/";
$target_file = $target_dir . basename($_FILES["fileToUpload"]["name"]);
$uploadOk = 1;
$imageFileType = pathinfo($target_file,PATHINFO_EXTENSION);
// Check if image file is a actual image or fake image
if(isset($_POST["submit"])) {
    $check = getimagesize($_FILES["fileToUpload"]["tmp_name"]);
    if($check !== false) {
        echo "File is an image - " . $check["mime"] . ".";
        $uploadOk = 1;
    } else {
        echo "File is not an image.";
        $uploadOk = 0;
    }
}
// Check if file already exists
if (file_exists($target_file)) {
    echo "Sorry, file already exists.";
    $uploadOk = 0;
}
// Check file size
if ($_FILES["fileToUpload"]["size"] > 500000) {
    echo "Sorry, your file is too large.";
    $uploadOk = 0;
}
// Allow certain file formats
if($imageFileType != "jpg" && $imageFileType != "png" && $imageFileType != "jpeg"
&& $imageFileType != "gif" ) {
    echo "Sorry, only JPG, JPEG, PNG & GIF files are allowed.";
    $uploadOk = 0;
}
// Check if $uploadOk is set to 0 by an error
if ($uploadOk == 0) {
    echo "Sorry, your file was not uploaded.";
// if everything is ok, try to upload file
} else {
    if (move_uploaded_file($_FILES["fileToUpload"]["tmp_name"], $target_file)) {
        echo "The file ". basename( $_FILES["fileToUpload"]["name"]). " has been uploaded.";
    } else {
        echo "Sorry, there was an error uploading your file.";
    }
}
?>

PHP5 Cookies

Cookie是用来区分用户的.
每次用户访问服务器的时候, 服务器会给用户的电脑嵌入一个文件, cookie. 当同一台电脑请求访问页面的时候, 电脑也会给服务器发送一个cookie.

Create Cookies With PHP

PHP中, 开发者可以用setcookie()来生成一个cookie

setcookie(name, value, expire, path, domain, secure, httponly);

PHP Create/Retrieve a Cookie

setcookie($cookie_name, $cookie_value, time() + (86400 * 30), "/");

  • 生成了一个叫做"user"的cookie, 值是John Doe.
  • 这个Cookie会在 30天后过期.
  • "/"表示这个cookie在整个网站都可以使用

开发者可以用$_COOKIE来检索"user"的值, 还可以用isset()来检查cookie是否被设置了.
注意: setcookie()必须在放置在<html>之前, cookie的值在上传时候会被自动的加密, 同时下载的时候也会被自动的解密. 如果要防止被加密的话, 可以使用setrawcookie().

<?php
$cookie_name = "user";
$cookie_value = "John Doe";
setcookie($cookie_name, $cookie_value, time() + (86400 * 30), "/"); // 86400 = 1 day
?>
<html>
<body>

<?php
if(!isset($_COOKIE[$cookie_name])) {
    echo "Cookie named '" . $cookie_name . "' is not set!";
} else {
    echo "Cookie '" . $cookie_name . "' is set!<br>";
    echo "Value is: " . $_COOKIE[$cookie_name];
}
?>

</body>
</html>

Modify a Cookie Value

如果要修改cookie的值, 还是使用setcookie()

Delete a Cookie

如果需要删除cookie, 只需要把截止日期设置为过去的时间即可

<?php
// set the expiration date to one hour ago
setcookie("user", "", time() - 3600);
?>
<html>
<body>

<?php
echo "Cookie 'user' is deleted.";
?>

</body>
</html>

Check if Cookies are Enabled

开发者可以用$_COOKIE来检测cookie是否可以用

<?php
setcookie("test_cookie", "test", time() + 3600, '/');
?>
<html>
<body>

<?php
if(count($_COOKIE) > 0) {
    echo "Cookies are enabled.";
} else {
    echo "Cookies are disabled.";
}
?>

</body>
</html>

PHP 5 Sessions

Session是用来提供多个页面之间传输数据的方法.
与Cookie不同的是, session不会存储在用户的电脑上.
Session的数据不会丢失 直到当用户关闭浏览器

Start a PHP Session

PHP中用seesion_start()来建立一个session, 会被存储到$_SESSION中.
注意session_start()一定要在文档的最开始, 尤其在<html>之前

<?php
// Start the session
session_start();
?>
<!DOCTYPE html>
<html>
<body>

<?php
// Set session variables
$_SESSION["favcolor"] = "green";
$_SESSION["favanimal"] = "cat";
echo "Session variables are set.";
?>

</body>
</html>

Get PHP Session Variable Values

Session不会自己传递到每个页面, 需要开发者在每个页面开头检索一下是否有session, 并且恢复session.
通常, session会在用户的电脑里面安置一个user-key, 当用户打开别的页面的时候, 它会搜寻这个user-key, 如果符合了, 就允许访问这个session, 如果不符合就会开始一个新的session.

<?php
session_start();
?>
<!DOCTYPE html>
<html>
<body>

<?php
// Echo session variables that were set on previous page
echo "Favorite color is " . $_SESSION["favcolor"] . ".<br>";
echo "Favorite animal is " . $_SESSION["favanimal"] . ".";
?>

</body>
</html>

Modify a PHP Session Variable

如果开发者要修改session, 那么只需要覆盖它即可

<?php
session_start();
?>
<!DOCTYPE html>
<html>
<body>

<?php
// to change a session variable, just overwrite it 
$_SESSION["favcolor"] = "yellow";
print_r($_SESSION);
?>

</body>
</html>

Destroy a PHP Session

如果需要移除所有的global session variables或者销毁这个session, 可以用session_unset()和session_destroy()

<?php
session_start();
?>
<!DOCTYPE html>
<html>
<body>

<?php
// remove all session variables
session_unset(); 

// destroy the session 
session_destroy(); 
?>

</body>
</html>
