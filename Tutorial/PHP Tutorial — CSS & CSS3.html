CSS

CSS 全名叫做叠层条样式表(Cascading Style Sheets)

  • CSS定义了HTML元素展示的样子
  • CSS有内嵌和外包两种形式
  • 由于CSS可以定义一类元素的样式, 所以节省了很多时间和省去了很多重复的代码

CSS Syntax

与HTML一样, CSS也不属于编程语言, 所以它的语法也很简单包含了 Selector 和 Declaration

h1 {color:blue; font-size:12px}
  • Selector (h1) 是选择开发者要修改样式的元素, 可以是一个特定的元素(id), 也可以是一类元素(class)
  • Declaration 则是开发者想要怎么去修改这个元素, 一个Declaration包含了一个property name 和对应的 value, 每个declaration都用分号进行分割

Selector

Element Selector

开发者可以选择某一种元素进行统一的修改样式

p {
    text-align: center;
    color: red;
}

id Selector

对于某个有id的元素, 开发者可以通过id selector来选中并且修改样式. 对于一个页面来说, id是唯一的.

#para1 {
    text-align: center;
    color: red;
}
<p id="para1">hello world</p>

class Selector

class Selector是用来选择一类元素来修改样式, 然而class是可以重复的, 所以选择的时候要注意是选择某一元素的class还是选择整个页面的所有class的元素

.center {
    text-align: center;
    color: red;
}
p.center {
    text-align: center;
    color: red;
}
<h1 class="center">Hello</h1>
<p class="center">World</p>

Child Selector

当某一个元素内部有另一个元素, 那么被包括的元素称之为外部元素的子元素, 那么开发者看而已通过外部元素 > 子元素 来对满足条件的子元素进行修改

div > p {
    background-color: yellow;
}

Adjacent Sibling Selector

相邻选择符的要求要是紧紧挨着的元素, 同时他们的父元素也要相同

div + p {
    background-color: yellow;
}

General Sibling Selector

相对于 Adjacent Sibling Selector, General Sibling Selector的要求比较宽松, 只要附近满足这个条件即可

div ~ p {
    background-color: yellow;
}

CSS [attribute] Selector

开发者可以用[attribute]来选择有特定attribute的元素

a[target] {
    background-color: yellow;
}

CSS [attribute=”value”] Selector

[attribute="value"] Selector可以更加精确的选择某个attribute的特定value.

a[target="_blank"] { 
    background-color: yellow;
}

CSS [attribute~=”value”] Selector

[attribute~="value"]Selector更可以精确到value的某个单词

[title~="flower"] {
    border: 5px solid yellow;
}

CSS [attribute|=”value”] Selector

[attribute|="value"]Selector可以用来选择attribute是带有特殊value开头的元素, 必须是完成的单词

[class|="top"] {
    background: yellow;
}

CSS [attribute^=”value”] Selector

[attribute^="value"]Selector可以用来选择attribute是带有特殊value开头的元素, 可以不是完整的单词

[class^="top"] {
    background: yellow;
}

CSS [attribute$=”value”] Selector

[attribute$="value"]Selector可以用来选择attribute是带有特殊value结束的元素, 可以不是完整的单词

[class$="test"] {
    background: yellow;
}

CSS [attribute*=”value”] Selector

[attribute*="value"]Selector可以用来选择attribute是含有特殊value的元素

[class*="te"] {
    background: yellow;
}

CSS的嵌入方式

External Style Sheet

当开发者写一些很大的网页的时候, 外包CSS是一个很好的方法 可以让开发者在多个页面公用一个CSS文件 这样可以省去很多重复的代码.
CSS文件是以.css结尾

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>
//main.css 文件
body {
    background-color: lightblue;
}

h1 {
    color: navy;
    margin-left: 20px;
}

Internal Style Sheet

内嵌CSS是位于HTML的<head><style>...</style></head>之间的, 这样做的好处是很清楚简单的把所有的CSS样式都集中在一起, 便于修改和debug. 

<head>
<style>
body {
    background-color: linen;
}

h1 {
    color: maroon;
    margin-left: 40px;
} 
</style>
</head>

Inline Styles

Inline Styles是目前用的较少的, 因为只会修改本元素的样式, 而且代码不可以重复使用, 所以不推荐这么写

<h1 style="color:blue;margin-left:30px;">This is a heading.</h1>

Cascading order

由于CSS有三种形式, 所以对于CSS各种形式的优先度也有不同(由低到高:)

  • 浏览器内置样式
  • External 和 Internal Style Sheets
  • Inline Style

优先度高的会覆盖优先度低的样式

CSS Background

CSS给HTML的页面增加了很多背景的样式, 包括颜色的修改, 添加图片作为背景 已经背景的循环…

Background Color

颜色的写法与HTML中一样, 包括16进制写法#ff00ff RGBrgb(255,0,0) 和 颜色名称red.

body {
    background-color: #b0c4de;
}

Background Image

CSS可以把整张图片设为背景, 支持目前大多数图片格式 包括png, jpg, gif…

body {
    background-image: url("paper.gif");
}

如果页面过大, 也可以让图片在水平repeat-x或者垂直repeat-y方向上反复显示, 当然也可以要求不重复现实no-repeat

body {
    background-image: url("gradient_bg.png");
    background-repeat: repeat-x;
}

Background - Shorthand property

由于CSS的background的property有很多, 所以有一种简单的写法, 只需要声明background, 然后把所有的value都写在一起, 用空格分隔即可
当然Shorthand property也是有顺序的, 依次为:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
body {
    background: #ffffff url("img_tree.png") no-repeat right top;
}

CSS Font Size

在CSS中设置的字体大小分为绝对尺寸和相对尺寸

  • 绝对尺寸
    • 字体的大小是特定的尺寸
    • 在所有的浏览器中, 用户都不能修改字体的尺寸
    • 当页面输出大小不清楚的话, 绝对尺寸是很有优势的
  • 相对尺寸
    • 尺寸会随着周围的元素进行相应的调整
    • 浏览器中, 用户可以改变字体的大小

Pixel 与 em

  • Pixel
    • 最普通的设置方法, 1px代表屏幕上一个像素
    • 当用户放大或者缩小页面的话, 字体会相应的变化
h1 {
    font-size: 40px;
}

h2 {
    font-size: 30px;
}

p {
    font-size: 14px;
}
  • em
    • W3C推荐的尺寸单位
    • 用户可以改变字体大小
    • 1em等于目前的字体尺寸, 默认的浏览器字体大小是16px, 所以默认的1em = 16px
h1 {
    font-size: 2.5em; /* 40px/16=2.5em */
}

h2 {
    font-size: 1.875em; /* 30px/16=1.875em */
}

p {
    font-size: 0.875em; /* 14px/16=0.875em */
}

CSS Link

在CSS中link的样式也很多, 开发者可以通过修改他们的style来美化link

  • a:hover - a link when the user mouses over it
  • a:active - a link the moment it is clicked

    color: #00FF00;

    color: #FF00FF;

}

CSS Position Property

CSS中元素的位置有4种形式: static, relative, fixed和absolute.

position: static

HTML的元素默认的位置属性就是static
static不会受到top, bottom, left和right 的影响

div.static {
    position: static;
    border: 3px solid #8AC007;
}

position: relative

当一个元素被设为relative, 那么它的位置会相对于它原来的位置发生改变
relative会受到top, bottom, left和right 的影响

div.relative {
    position: relative;
    left: 30px;
    border: 3px solid #8AC007;
}

position: fixed

当一个元素被设为fixed, 那么它的位置不会随着页面的滚动发生改变, 因为它的位置是固定于视窗(也就是浏览器的窗口).
如果随着页面的滚动, 这个元素的位置下有别的元素, 那么会导致这个元素覆盖住别的元素

div.fixed {
    position: fixed;
    bottom: 0;
    right: 0;
    width: 300px;
    border: 3px solid #8AC007;
}

position: absolute

当一个元素被设为absolute, 那么它的位置会固定于最近的元素(nearest positioned ancestor)的位置.
如果这个元素没有最近的元素, 那么它就会固定于整个页面, 并且随着页面的滚动而移动

div.relative {
    position: relative;
    width: 400px;
    height: 200px;
    border: 3px solid #8AC007;
} 

div.absolute {
    position: absolute;
    top: 80px;
    right: 0;
    width: 200px;
    height: 100px;
    border: 3px solid #8AC007;
}

Overlapping Element

CSS中可以通过设置z-index来修改元素之间的上下层级的位置关系, 在上层级的元素会覆盖下层级的元素

img {
    position: absolute;
    left: 0px;
    top: 0px;
    z-index: -1;
}

CSS Float Property

简单的来说, float可以实现让文字围绕图片的效果

CSS Clear Property

clear可以用来控制float元素的行为
一般来说, float元素之后的元素会包围它, 为了避免这种情况, 开发者可以使用clear

如果float元素大于它的container, 那么它会超出并且覆盖这个container, 而overflow:auto可以解决这个问题

.clearfix {
    overflow: auto;
}

.floating-box {

CSS Pseudo-elements

CSS中 可以用Pseudo-element来修改一个元素的特定部分 比如:

Navigation Bars

开发者可以用CSS来建立一个最简单的导航栏, 其实也就是一串链接

<!DOCTYPE html>
<html>
    <head>
        <style>
            ul {
                list-style-type: none;
                margin: 0;
                padding: 0;
                overflow: hidden;
            }

            li {
                float: left;
            }

            a {
                display: block;
                width: 60px;
                background-color: #dddddd;
            }
        </style>
    </head>
    <body>
        <ul>
          <li><a href="#home">Home</a></li>
          <li><a href="#news">News</a></li>
          <li><a href="#contact">Contact</a></li>
          <li><a href="#about">About</a></li>
        </ul>
        <p><b>Note:</b> If a !DOCTYPE is not specified, floating items can produce unexpected results.</p>
        <p>A background color is added to the links to show the link area. The whole link area is clickable, not just the text.</p>
        <p><b>Note:</b> overflow:hidden is added to the ul element to prevent li elements from going outside of the list.</p>
    </body>
</html>

The justify-content Property

The justify-content property horizontally aligns the flexible container’s items when the items do not use all available space on the main-axis.
The possible values are as follows:

  • flex-start - Default value. Items are positioned at the beginning of the container
  • flex-end - Items are positioned at the end of the container
  • center - Items are positioned at the center of the container
  • space-between - Items are positioned with space between the lines
  • space-around - Items are positioned with space before, between, and after the lines

    display: -webkit-flex;
    display: flex;
    -webkit-justify-content: flex-end;

The align-items Property

The align-items property vertically aligns the flexible container’s items when the items do not use all available space on the cross-axis.
The possible values are as follows:

  • stretch - Default value. Items are stretched to fit the container
  • flex-start - Items are positioned at the top of the container
  • flex-end - Items are positioned at the bottom of the container
  • center - Items are positioned at the center of the container (vertically)
  • baseline - Items are positioned at the baseline of the container

The flex-wrap Property

The flex-wrap property specifies whether the flex items should wrap or not, if there is not enough room for them on one flex line.
The possible values are as follows:

  • nowrap - Default value. The flexible items will not wrap
  • wrap - The flexible items will wrap if necessary
  • wrap-reverse - The flexible items will wrap, if necessary, in reverse order

    display: flex;
    -webkit-flex-wrap: wrap-reverse;
    flex-wrap: wrap-reverse;

    height: 250px;
    background-color: lightgrey;
}

The align-content Property

The align-content property modifies the behavior of the flex-wrap property. It is similar to align-items, but instead of aligning flex items, it aligns flex lines.
The possible values are as follows:

  • stretch - Default value. Lines stretch to take up the remaining space
  • flex-start - Lines are packed toward the start of the flex container
  • flex-end - Lines are packed toward the end of the flex container
  • center - Lines are packed toward the center of the flex container
  • spa`ce-between - Lines are evenly distributed in the flex container
  • space-around - Lines are evenly distributed in the flex container, with half-size spaces on either end
